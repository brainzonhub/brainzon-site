# Brainzon Design System


## Brand Style

Enterprise Software + AI + Digital Transformation.

Visual keywords:

- Intelligent
- Modern
- Secure
- Global
- Enterprise


## Theme Architecture

Implement three themes:

system (default)
dark
light


Use CSS variables:

--background
--foreground
--card
--card-border
--primary
--secondary
--accent
--muted


## Dark Theme

Primary appearance:

Background:
Deep navy / black

Example:
#020817

Accent:
Technology cyan

Highlight:
Brainzon red/orange


## Light Theme

Professional enterprise style:

- White backgrounds
- Soft gray surfaces
- Dark typography
- Brand accent colors


## Typography

Use:

Geist / Inter

Headings:
Bold, large, enterprise style

Body:
Readable and clean


## Logo Color Usage

Extract colors from Brainzon logo.

Primary brand colors:

- Red
- Cyan/Blue
- Yellow accent


## Mega Menu

Desktop navigation:

Products
Solutions
Modernization
Company
Resources


Products mega menu:

BrainERP
Enterprise Resource Planning

BrainCRM
Customer Relationship Management

BrainAI
Enterprise AI Platform

BrainConnect
Integration Platform


Each item should include:
- Icon
- Description
- Link


## UI Elements

Cards:

- Rounded corners
- Glass effect
- Subtle borders
- Hover glow


Buttons:

Primary:
Brand gradient

Secondary:
Outline style


## Motion

Use Framer Motion:

- Fade
- Slide
- Glow transitions

Keep animations professional.
