# Development Guidelines


## Theme

Install:

next-themes


Provider:

ThemeProvider

attribute:
class

defaultTheme:
system


## Tailwind

Use:

darkMode: class


Prefer:
Tailwind utilities

Avoid unnecessary CSS.


## TypeScript

Never use:

any


Use:
interfaces
types
generics


## SEO

Every page requires:

- Metadata
- OpenGraph
- Twitter cards
- Structured data


## Performance

Use:

next/image
next/font

Optimize:
- Images
- Loading
- Bundle size


## Components

Keep components:

- Small
- Reusable
- Maintainable
