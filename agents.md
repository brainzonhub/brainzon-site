# Brainzon Website - Codex Agent Instructions (Multipage Version)

## Project Vision

Build the official Brainzon enterprise website as a production-grade multi-page digital platform.

Brainzon positioning:

"Digital Transformation Partner for Modern Businesses"

The website must present Brainzon as a global software company providing:

- Proprietary SaaS Products
- Enterprise ERP Solutions
- CRM Platforms
- AI Solutions
- Integration Platforms
- Legacy Application Modernization
- Cloud Transformation Services


# Technology Stack

Use:

- Latest stable Next.js
- TypeScript
- Tailwind CSS
- App Router
- React Server Components
- next-themes for theme management


# Application Type

This is a MULTI-PAGE enterprise website.

Do not build a single landing page.

Each major business area requires its own page, reusable sections, SEO metadata, and structured content.

# Development Rules

Always:

- Use strict TypeScript
- Create reusable components
- Avoid duplicated layouts
- Use data-driven page sections
- Add SEO metadata for every page
- Optimize images
- Follow accessibility standards


# SEO Requirements

Every route must include:

- Metadata
- OpenGraph information
- Structured data
- Proper headings hierarchy


# Code Quality

Do not create:

- Temporary code
- Hardcoded repeated content
- Large monolithic components

Build like a global enterprise software company website.
